## Como Usar

1. Clone o repositório: git clone https://github.com/ryannardelli/android.git
2. Abra o arquivo 'index.html' no seu navegador

## Instalação

Não é necessário instalação. Basta abrir o arquivo HTML no navegador da sua escolha.
